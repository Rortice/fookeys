Pinia呼び出しのとき
storeToRefs(store) // storeの中身をrefに変換する
toRefs(store) // storeの中身をrefに変換する
の仕様がいまいちわかっていない

//現在
const { player } = storeToRefs(playerStore)
const { idGame } = toRefs(player.value)
idGame.value // これで値を取得できるし､リアクティブになる

//他の書き方1
const { player } = storeToRefs(playerStore)
const { idGame } = player.value
idGame // これで値を取得できるかも?

//他の書き方2
const { player } = playerStore
player.idGame // これで値を取得できるが､リアクティブではない

仮説1 分割代入はリアクティブにならない ⭕
仮説2 分割代入した値に代入ができない
